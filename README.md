# WithU

충북학사 서서울관 학사생활 도우미 앱

함께에유~ => With유~ => WithU

한지붕 한가족인 충북학사의 정겨움을 표현한 이름.

## 기능

기존 학사 웹페이지에서 분산화되고 불편한 서비스를 하나의 앱으로 제공합니다.

외박 신청, 식단, 재사요건 확인, 상/벌점 확인, 입/출입 QR 코드 발급
